# 策略模式

在策略模式（`Strategy Pattern`）中，一个类的行为或其算法可以在运行时更改。这种类型的设计模式属于**行为型模式**。

在策略模式中，我们创建表示各种策略的对象和一个行为随着策略对象改变而改变的 `context` 对象。策略对象改变 `context` 对象的执行算法。

意图：定义一系列的算法, 把它们一个个封装起来, 并且使它们可相互替换。
关键代码：**实现同一个接口**。

策略模式能体现出`JAVA`特性中的**多态**和设计原则中的**开闭原则**
