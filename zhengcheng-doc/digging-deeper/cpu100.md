# 线上服务 CPU 100% 的处理过程 :hammer:

## 解决步骤

1. 执行`top`命令查看，定位到占用`CPU`高的进程`ID`，按一下`P`可以按照`CPU`使用率进行排序
```shell script
top -c
```

2. 使用 `top -Hp <进程ID>` 定位到占用`CPU`高的线程`ID`，继续按`P`排序
```shell script
top -Hp <进程ID>
```

3. 将线程`ID`转换成为16进制，为后面查找`jstack`日志做准备
```shell script
printf "0x%x" 线程PID
```

4. `jstack` 进程`PID` | vim +/`十六进制线程PID`
